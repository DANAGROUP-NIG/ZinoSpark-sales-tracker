## ğŸ”§ Environment Variables

### Configuration
The app uses environment variables for API configuration:

```bash
# .env.local (Development)
NEXT_PUBLIC_API_BASE_URL=http://localhost:3001/api

# .env.production (Production)
NEXT_PUBLIC_API_BASE_URL=https://api.zinospark.com/api
```

### Available Variables
- `NEXT_PUBLIC_API_BASE_URL`: Base URL for API endpoints
  - Development: `http://localhost:3001/api`
  - Production: `https://api.zinospark.com/api`

### Setup Instructions
1. Copy `.env.example` to `.env.local`
2. Update the API base URL for your environment
3. For production, set the appropriate URL in your deployment platform
